ESP Simulation

By Peter Gillich

Requires

  • Latest Sun/Oracle JDK, perhaps JRE is enough. Tested with JDK 6 Update 24 (1.6.24).
  • Enabled Sun/Oracle Java Plug-in in the web browser. It's enabled by default.
  • Java 3D. After install, browsers must be restarted. Old video cards aren't supported. Tested with Java 3D 1.5.2 (Windows).

Model Description

It's a very simple ESP simulation. See more about ESP on Wikipedia: Electronic Stability Control. See more details about the model in a Mathcad 14 file: ESP.xmcd.

Usage

After start, select the Model tab and change some parameter, for example:

  • CorrectingBreakEnaled: Enables/disables correcting break.
  • Dt: Delta t, smalles means slower simutation.
Press the Play button. On the Display window, 3D operations (zoom, pan, rotation) are possible by left/middle/right button + mouse movement.
